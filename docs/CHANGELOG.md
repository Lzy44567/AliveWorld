# 更新日志

## [v1.1.0] - 2026-07-11 - 切片 18：全局默认与局内设置持久化

作者/环境：`codex-gpt5.6sol`

### 新增与修复

- 新增独立的存档设置模型：全局设置作为新故事默认模板，创建时复制；已有存档之后独立修改。
- 局内设置覆盖未来可能性、掷骰、重掷、AI 建议、实体推演、实体显示权限和顶部时间显示。
- “宇宙主导向简述”与“主线剧情导向”拆成 `world_premise`、`plot_compass` 两个独立字段，后者不再覆盖世界前提。
- 宇宙主导向新增局内编辑入口；主线剧情导向同时进入随机反应和正文结算上下文。
- 实体库顶部增加存档级暗流推演快捷开关，与局内设置共享同一个持久化状态。
- 旧存档的 `description` 自动迁移为 `world_premise`，缺失的局内设置使用安全默认值补齐。
- 局内设置提供“恢复全局默认”与显式保存，API 连接信息仍只属于全局环境。

### 验证

- 新增设置白名单归一化、两种导向分离持久化和旧存档迁移测试。

## [v1.1.0] - 2026-07-11 - 切片 17：Overseer 空跑门控与设置分区

作者/环境：`codex-gpt5.6sol`

### 新增与修复

- 增加“启用暗流实体推演”总开关；关闭时保留实体卡和逐卡状态，但不请求 Overseer。
- 后端在实体库为空或所有实体均封存时自动跳过 Overseer，不消耗对应 token，也不增加该阶段等待时间。
- 设置页默认打开“调试与推演”，API 配置移至第二项；侧栏现在可以真实切换内容。
- 设置页拆为推演设置、API 设置、实体显示设置三个独立组件，模态框只负责统筹切换。
- 行动、重试、重掷与世界初始化统一传递实体推演总开关。

### 验证

- 增加空实体、全封存、总开关关闭均不得调用 AI 的后端测试。
