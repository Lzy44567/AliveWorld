## [v1.1.0] - 2026-07-10 - 切片 2：Overseer 协议升级

作者/环境：`codex-gpt5.6Terra`

### 变更

- Overseer 现在明确可以返回空动作，不再要求每回合制造暗流事件。
- 默认统筹者不再称为“世界意志”；“天道/世界意志”只在世界观真实存在时作为独立实体。
- 事件协议支持状态、增量计划、持续机制、未来触发线索、关系变化和主线可用线索。
- 模型只接收活跃实体；即使返回封存或不存在实体的动作、更新、删除，也会被后端忽略。
- 影子账本保存结构化依据，并仅投影有限线索给主线上下文。

### 验证

- `system_prompts.yml` YAML 解析通过。
- `python -m unittest discover -s tests -v`：4 项通过，包含封存实体拒绝执行的行为测试。
- `npm run build` 通过。

### 已知范围

- 尚未提供“世界推演”实体模板或 UI 可见性选择；这是后续切片，不影响本协议使用现有实体卡。
