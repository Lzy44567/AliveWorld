# 📖 AliveWorld : AI 动态推演引擎开发总纲 (v3.6)

## 一、 项目架构演进
1. **推演双轨与 Tick 引擎：** AI 发散 -> Python 掷骰 -> AI 生成 -> Python 结算 Buff。
2. **模块中枢 (Tavern)：** 角色卡、文风卡、世界书全部解耦为独立文件。
3. **动态记忆与注入 (RAG 雏形)：** 采用“全局常量 + 关键词触发词条”的酒馆世界书模式，解决大模型注意力涣散问题。
