# 📖 AliveWorld : AI 动态推演引擎开发总纲 (v3.8)

## 一、 项目架构演进
1. **推演双轨与 Tick 引擎：** AI 发散 -> Python 掷骰 -> AI 生成 -> Python 结算 Buff。
2. **模块中枢 (Tavern)：** 角色卡、文风卡、世界书全部解耦为独立文件。
3. **动态记忆与注入：** 采用“全局常量 + 关键词触发词条”的酒馆世界书模式。
4. **【New】AI辅助创世机制：** 引入大语言模型作为内容辅助生产力，支持生成整本世界书或单一词条，降低剧本准入门槛。
