\# AliveWorld 更新日志



\## \[v3.6.0] - 核心推演与酒馆架构整合版

\### 新增 (Added)

\* 实现了 AI 发散、Python 掷骰、AI 结算的双轨推演引擎。

\* 独立了解耦的模块中枢，支持读取 YAML 格式的角色卡、文风卡、世界书。

\* 实现了世界书动态词条扫描机制（基于关键词触发）。

\### 待修复 (To Fix)

\* 主干对话流中的自动保存逻辑存在遗漏，导致 API 异常时丢失进度。

\* 大厅系统的“文风工坊”缺少对应的 UI 代码块实现。



\## \[v3.7.0] - AI 辅助创世与词条铸造版

\### 新增 (Added)

\* \*\*AI 创世神机制\*\*：在世界书工坊加入“一键生成世界观”功能。

\* \*\*AI 词条铸造机\*\*：新增对单一词条的独立扩展功能，玩家只需输入短句灵感，AI自动扩写为详尽的法则/背景说明。

\* \*\*开发蓝图升级\*\*：明确了下一步“向量化语义识别（RAG）”的开发路径。



\### 修复 (Fixed)

\* 修复了部分终端由于 `engine.py` 未正确加载导致的 `generate\_worldbook` 方法缺失报错。

\* 优化了“文风大师”的草稿留存逻辑，防止页面刷新导致草稿丢失。



\## \[v4.0.0] - 专业日志监控与异常拦截护盾 (The Ironclad Update)

\### 新增 (Added)

\* \*\*独立专业级日志视图\*\*：引入完美复刻 IDE 终端的网页端全屏日志视图，支持模块来源、图标分类、时间戳与原声 JSON 展示。

\* \*\*GlobalCatcher 全局异常拦截网\*\*：在 `app.py` 外层包裹了防溃系统。任何内部变量缺失、AI幻觉导致的报错，都不会再让网页白屏或闪退，而是会引导玩家进入“抢修模式”查看堆栈信息。

\* \*\*大厅代码解耦重构\*\*：将所有 Lobby 渲染逻辑从 `app.py` 彻底抽离至 `ui\_tavern.py` 的 `render\_tavern` 方法中，代码复杂度下降 40%。



\## \[v4.0.1] - 核心 JSON 引擎约束修复版

\### 修复 (Fixed)

\* 修复了由于 API 缺失 `response\_format={"type": "json\_object"}` 导致大模型偶发性输出纯小说文本（跳过JSON）的严重幻觉问题。

\* 修复了由于 `max\_tokens` 设置过紧导致长剧情或长 JSON 结算时发生文本截断的 Bug。目前已放宽至基础 2500 tokens。



\## \[v4.0.2] - API 虚空审查防溃盾

\### 修复 (Fixed)

\* 修复了因用户输入高强度 NSFW 内容触发大语言模型“安全内容过滤器（Content Filter）”导致返回空字符串，进而引发 JSON 挖掘彻底崩溃的致命 Bug。

\* 引入了\*\*【虚空兜底机制】\*\*：当检测到 API 返回静默空串时，程序不再崩溃，而是会在 UI 输出“系统提示：触发 API 审查”，保障当前存档的因果线不断裂。
