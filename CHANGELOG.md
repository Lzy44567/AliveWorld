# AliveWorld 更新日志 (Changelog)

## [v1.0.0] - 创世正式版 (Formal Release) - 2026-06-XX
*经过了漫长的 MVP 概念验证与底层重构，AliveWorld 迎来了真正的 V1.0.0 正式版。此版本彻底抛弃了早期硬编码的桎梏，确立了“高自由度动态沙盒”与“暗流双轨推演”的终极架构。*

### 🚀 史诗级核心重构
- **动态物理沙盒引擎**：彻底剥离了死板的 HP/MP 等硬编码属性。一切数值、进度条均由 AI 动态接管并下发，完美支持修仙、赛博、废土等无限世界观的自由切换。
- **暗流实体真实降临**：重写了 `UndercurrentEngine`。 Overseer（世界意志）推演出的暗中实体、势力变化，将直接实时以 `.yml` 物理文件的形式写入本局沙盒存档，真正实现“万物皆有因果，实体皆可追溯”。
- **Vue3 模块化重构**：前端代码彻底告别“屎山”。解耦拆分了雷达面板、状态条渲染、多重弹窗管理，引入了极致的色彩智能自适应系统，UI 表现力与拓展性产生质的飞跃。

### ✨ 新增功能特性
- **双向资产流通体系**：实现了“全局万象图鉴”与“局内专属设定”的物理隔离与双向另存为机制。玩家可随时向世界中拖入新角色、世界书，或将本局衍化出的极品 NPC 推送回全局图鉴。
- **命运观测与完美重掷**：新增 N*n 未来观测包面板。接入独立的 `/reroll` 接口，支持玩家在不满意命运分支时，通过强制排异算法，平滑地重选一条全新的时间线。
- **智能 Buff 与环境解析**：全面支持基于回合制的持续状态结算（如中毒、自愈）。前端新增极具沉浸感的高亮色标签，能够智能识别 NPC 阵营并赋予相应的色彩表现。

### 🔧 体验优化与修复
- 实现了前后端大模型 API 配置的实时热重载。
- 修复了回溯记忆时，物理沙盒实体文件未能同步回档的“幽灵实体”灾难性 Bug。
- 增加了极其醒目的时间流逝置顶显示，并支持在局内设置中自由开关环境感知 UI。
